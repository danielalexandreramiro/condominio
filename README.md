# Condomínio App

Sistema web para visualização de balancetes financeiros de condomínio, desenvolvido com Angular.

---

# Funcionalidades

- Visualização de balancete mensal
- Resumo financeiro das contas
- Receitas e despesas detalhadas
- Cards de resumo financeiro
- Responsividade para celular
- Scroll horizontal automático em tabelas grandes
- Interface moderna e limpa
- Gráficos financeiros
- Filtros por mês

---

# Tecnologias

- Angular 19
- TypeScript
- SCSS
- Bootstrap
- Chart.js

---

# Responsividade

O sistema foi ajustado para funcionar corretamente em dispositivos móveis:

- Cards adaptáveis
- Tabelas com scroll horizontal
- Layout otimizado para celular
- Gráficos responsivos

# Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

Entre na pasta:

```bash
cd seu-repositorio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
ng serve
```

Abra no navegador:

```bash
http://localhost:4200
```

---

# Build

```bash
ng build
```

Build de produção:

```bash
ng build --configuration production
```
